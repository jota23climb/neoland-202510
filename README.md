## Functional description

### Use cases

User

- register
- login
- update credentials (username, password)
- update profile (name, email, phone, city, role, licensing number)

Owner (User)
- add pet
- remove pet
- modify pet
- list pets
- add comment log for pet
- modify comment log for pet
- list logs for pet

Veterinary (User)
- list assigned pets
- add action log for pet
- list logs for pet
